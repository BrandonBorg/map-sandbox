# app

This folder contains the React + TypeScript frontend for the map sandbox.

## Purpose

The app is used to render and experiment with map-based interfaces connected to the tile server. It is a lightweight place to try different map visuals, layers, and UI patterns.

## Stack

- React
- TypeScript
- Vite
- MapLibre GL

## Development

Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

## Build

To create a production build:

```bash
npm run build
```
