# Source Data

This folder contains the source geospatial datasets used by the map sandbox project.

## Contents

- ODB_v3/ - Ontario and Canada boundary dataset files in GeoPackage format

## Notes

- Files in this folder are source datasets and are not generated by the application.
- GeoPackages and GeoJSON files are intended for ingestion into the tile server pipeline.
- Keep large source files in this location and reference them from ingestion scripts.
- The ODB v3 files are git-ignored and can be downloaded from the Open Database of Buildings (ODB) portal: https://www150.statcan.gc.ca/n1/pub/34-26-0001/342600012018001-eng.htm

## Typical Usage

These files can be used to:
- test data ingestion workflows
- generate Parquet-backed H3 tiles
- validate map rendering and server responses
